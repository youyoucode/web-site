# 提示1：
# 要想顺利施放火球术，你需要装备“狂焰晶石”。
# 提示2：
# 采用循环语句，判断有敌人之后，先看看自己的火球术有没有冷却（hero.isReady()），如果有就放火球，否则直接攻击。

hero.moveXY(23,37)

while True:
    enemy = hero.findNearestEnemy()
    if hero.isReady("fireball"):
        hero.cast("fireball", enemy)
    else:
        hero.attack(enemy)
