诺瓦 Python Level2

本课程以闯关制结合项目制的教学方式，以游戏化的形式让学生掌握Python的基本语法，快速入门Python编程。

寓教于乐的教学方式，从被动学习转变为主动学习，让孩子爱上编程。


教学目标：

1. 掌握Python基本语法

2. 独立制作简单小游戏，小项目

3. 培养计算思维，抽象思维和解决分析问题能力

4. 掌握书写百行代码项目的开发能力，相当于大学计算机专业2年级水平



课程对象

本课程建议9岁以上学生学习

上课形式

48课时，小班制，真人在线直播，课后在线答疑

学习时长

6个月左右
